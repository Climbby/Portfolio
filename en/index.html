This is my first functional website and game, guessit, which started in March 2025. It is a game based on wordle, where you win if the secretly and randomly chosen player matches the player you entered.

When you enter a player, boxes with characteristics and a color are shown, where green means right, and red means wrong. You win the game when all the boxes turn green.
TheBridger is a project I started in April 2025, this is a chat-bot made with python and external libraries, TheBridger. TheBridger, which was made to be a bot on Discord, aims to simulate a game of The Bridge.

This bot was created because the mentioned game mode often lacks the minimum number of players required to start.

Your decisions matter! You have to make the right choices, which will change the probabilities of certain events happening, and you have to destroy the opponents' Nexus.
